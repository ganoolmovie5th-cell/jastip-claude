# CLAUDE.md

Steering file untuk Claude dan asisten berbasis Claude yang bekerja di repository ini.

Seluruh panduan utama ada di [`AGENTS.md`](AGENTS.md). Baca itu lebih dulu.

## Ringkas cepat

- Proyek: landing page statis jasa titip (jastip) bernama **Jastipin**.
- Stack: HTML + CSS + JavaScript vanilla. Tanpa build step, tanpa framework.
- Konversi bertumpu pada kepercayaan. Jaga transparansi harga, bukti proses, dan testimoni.
- Bahasa untuk pengguna: Bahasa Indonesia natural.
- Aturan keras: tanpa em-dash, satu sistem desain, aksesibilitas dijaga.
